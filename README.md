# johnniemacs

Yet another emacs configuration.

## Installation
Clone repo to your home dir.

```bash
git clone https://github.com/johnnie-che/johnniemacs ~/.emacs.d
```

Add this to the emacs config file (`~/.emacs`):
```elisp
(add-to-list 'load-path "~/.emacs.d/johnniemacs")
(require 'johnniemacs)
```
